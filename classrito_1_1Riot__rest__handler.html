Executes REST request synchronously.

Parameters
type- type of HTTP request (GET, POST, etc)
message- request query. Can be either just URL ("/riot/account/v1/accounts/me") or concatenated URL + params ("/some/url?param1=value1&param2=value2")
Exceptions
No_region_selected_exception
Unknown_exception
Message_exception
Wrong_enum_value_exception
Returns
HTTP response.
Precondition
Region has to be selected using set_region method.

Sends HTTP request and returns HTTP response from the server. If the region is not selected, throws No_region_selected_exception.

If provided Request_type has incorrect or unspecified value, Wrong_enum_value_exception is thrown.

If there is some error while sending request to the server or receiving response, either Message_exception or Unknown_exception is thrown (depending on source of error).
