Detailed Description

Handler for REST communication with LCU (League of Legends client).

Handles REST requests and responses to LCU synchronously. Automatically connects to LCU in the background.

Executes REST request synchronously.

Parameters
type- type of HTTP request (GET, POST, etc)
message- request query. Can be either just URL ("/lol-summoner/v1/current-summoner") or concatenated URL + parameters ("/some/url?param1=value1&param2=value2")
Exceptions

Unknown_exception
Message_exception

Returns
HTTP response.

Sends HTTP request and returns HTTP response from the server. Automatically connects to the LCU in the background using Lcu_process_handler class and its method Lcu_process_handler::get_lcu_process_parameters(), therefore potentially throws all exceptions that Lcu_process_handler::get_lcu_process_parameters() throws.

If provided Request_type has incorrect or unspecified value, Wrong_enum_value_exception is thrown.

If there is some error while sending request to the server or receiving response, either Message_exception or Unknown_exception is thrown (depending on source of error).
